# LingoArcade Frontend

This directory contains the React/Vite frontend for LingoArcade.

## Scripts

```powershell
npm run dev
npm run lint
npm run build
npm run preview
```

## Environment

Create `frontend/.env` for local development:

```text
VITE_API_BASE_URL=http://127.0.0.1:8000
```

For deployment, set `VITE_API_BASE_URL` in the hosting provider to the deployed FastAPI backend URL.
